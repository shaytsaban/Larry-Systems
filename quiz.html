Lesson 3 מפרסמים דשבורד ב-GitHub Pages
המרצה: לארי — כל החומר מבוסס על ההרצאות וההדרכה שלו

מ-Tableau אל האתר החי — כל המסלול, מקצה אל קצה.

סקירת מושגים, אינפוגרפיקה של תהליך העבודה, השוואה בין כלי AI, ו-18 שאלות עם משוב מיידי והסבר. מטרת היחידה: לוודא שאתם יודעים לקחת דשבורד שבניתם ולהטמיע אותו באתר אמיתי שמתארח בחינם ב-GitHub Pages — בלי שרת, בלי תחזוקה, בלי סיבוכים.

התחלת הבחינה סקירת מושגים תחילה
01 · Workflow

10 הצעדים — מהתוכנה המקומית אל האתר באוויר.

זהו המסלול המלא שנסקר בשיעור. כל צעד מוביל ישירות לצעד הבא.

TABLEAU

בניית הדשבורד

AI TOOL

02a · AI Tools Comparison

כלי AI — יתרונות וחסרונות.

השוואה בין כלים פופולריים ליצירת קוד HTML, חינמיים ובתשלום.

💬

ChatGPT

חינמי + פרימיום
יתרונות
  • חינמי עם גרסאות GPT-3.5/4o-mini
  • מודל 4o/4o-Turbo בתשלום — איכות גבוהה
  • קהילה גדולה, הרבה תבניות מוכנות
חסרונות
  • אין Preview מובנה — צריך להעתיק לסביבה חיצונית
  • הגרסה החינמית מוגבלת במספר הודעות/שעה
  • לעיתים מפספס הקשר בקבצים גדולים
🤖

Claude

חינמי + פרימיום
יתרונות
  • מודל 3.5 Sonnet — מצוין לכתיבת קוד
  • חלון הקשר גדול (200K טוקנים)
  • תשובות מנוסחות ומובנות היטב
חסרונות
  • אין Preview מובנה של ה-HTML
  • הגרסה החינמית מוגבלת בשימוש יומי
  • גרסאות מתקדמות דורשות מנוי בתשלום
🔍

DeepSeek / Qwen

חינמי
יתרונות
  • חינמי לחלוטין
  • מתמחה בקוד ופתרון בעיות טכניות
  • מודלים קוד-פתוח (Open Source)
חסרונות
  • אין תצוגה מקדימה (Preview) מובנית
  • לעיתים פחות מדויק בעיצוב ו-CSS
  • ממשק פחות אינטואיטיבי למתחילים
✖️

Grok

חינמי (עם חשבון X)
יתרונות
  • Preview מובנה — רואים את התוצר מיד
  • נגיש ישירות דרך grok.com
  • מומלץ בהדרכה בזכות קיצור תהליכים
חסרונות
  • דורש חשבון X (טוויטר)
  • גרסה חינמית מוגבלת במספר הודעות
  • פחות מוכר בקהילת המפתחים
👨‍🏫
לארי ממליץ: למתחילים — השתמשו ב-Grok בזכות ה-Preview המיידי שמקצר משמעותית את תהליך הפיתוח. למתקדמים — Claude 3.5 Sonnet או ChatGPT-4o נותנים תוצאות קוד איכותיות יותר, גם אם אין בהם Preview מובנה. כל הכלים החינמיים מספיקים למטרה זו.
02b · Key Concepts

המילון — 8 מושגים שחייבים לזכור.

לפני המבחן: רענון של המונחים החשובים ביותר מהחומר.

▲ TABLEAU

Tableau Public

פלטפורמת ענן של Tableau לאחסון ופרסום ויזואליזציות באופן חינמי. כל מה שנשמר אליה נגיש דרך public.tableau.com.

▲ TABLEAU

Copy Link / Copy Embed

Copy Link — קישור ישיר לדשבורד. Copy Embed — קטע HTML מוכן להטמעה. ההדרכה משתמשת ב-Copy Link.

▲ GITHUB

Repository

"מחסן" ייעודי לפרויקט ב-GitHub. מכיל את קבצי הקוד, HTML, ונכסים נוספים של הפרויקט.

▲ GITHUB

index.html

קובץ ברירת המחדל של האתר. שם הקובץ חייב להיות index.html כדי ש-GitHub Pages יציג אותו אוטומטית.

▲ GITHUB

Commit changes

פעולת "שמירה" ב-GitHub. מאשרת שינויים בקובץ ומעדכנת אותם במחסן. הכפתור הירוק בפינה.

▲ HOSTING

GitHub Pages

שירות אירוח סטטי חינמי. מאפשר להפוך כל Repository לאתר חי. מוגדר תחת Settings → Pages.

▲ GITHUB

main (branch)

הענף הראשי והדיפולטי. בהגדרות Pages בוחרים אותו כמקור לפריסת האתר.

▲ AI TOOLS

Preview מובנה

תכונה ייחודית (בעיקר ב-Grok) שמאפשרת לראות תוצאה חזותית מידית של ה-HTML שנוצר, מבלי לעזוב את הכלי.
